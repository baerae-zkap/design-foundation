Toast

사용자에게 피드백 메시지를 일시적으로 표시하는 알림 컴포넌트입니다.

Types

상황에 맞는 타입을 선택합니다. 버튼을 클릭해보세요.

// Toast 표시
showToast("저장되었습니다.", "success");
showToast("오류가 발생했습니다.", "error");
showToast("주의가 필요합니다.", "warning");
showToast("새로운 알림이 있습니다.", "info");
