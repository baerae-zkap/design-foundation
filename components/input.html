Input

사용자로부터 텍스트 입력을 받는 기본 폼 요소입니다. 다양한 타입과 상태를 지원합니다.

Basic

기본 입력 필드입니다. 직접 입력해보세요.

<Input
  label="이름"
  placeholder="이름을 입력하세요"
  value={value}
  onChange={setValue}
/>

Types

텍스트, 이메일, 비밀번호 타입을 지원합니다.

<Input type="email" label="이메일" />
<Input type="password" label="비밀번호" />

States

에러 및 비활성화 상태를 지원합니다.

필수 입력 항목입니다.
<Input error="필수 입력 항목입니다." />
<Input disabled />
