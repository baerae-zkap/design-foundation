Checkbox

사용자가 여러 옵션 중 하나 이상을 선택할 수 있는 선택 요소입니다.

Basic

기본 체크박스입니다. 클릭해보세요.

<Checkbox
  label="옵션 1"
  checked={checked}
  onChange={setChecked}
/>

States

체크됨, 체크 안됨, 비활성화 상태를 지원합니다.

Group Example

전체 선택 기능이 포함된 체크박스 그룹 예시입니다.

PropTypeDefaultDescription
