기본 그림자 단계입니다.

그림자 없음

none

shadowExtraSmall

elevation 1. 미세한 분리감

shadow2XLarge

0px 16px 32px rgba(16, 23, 38, 0.16)

Card

카드 컴포넌트용 그림자입니다.

cardDefault

기본 카드 그림자

cardElevated

강조 카드, 호버 상태 카드

cardFloating

플로팅 카드, 선택된 카드

Button

버튼 컴포넌트용 그림자입니다.

buttonDefault

기본 버튼은 그림자 없음

buttonElevated

elevated 버튼, FAB

buttonPressed

눌린 상태 버튼

Header & TabBar

헤더와 탭바 컴포넌트용 그림자입니다.

headerDefault

기본 헤더는 그림자 없음

headerScrolled

스크롤 시 헤더 그림자

tabBarDefault

기본 탭바는 그림자 없음

tabBarElevated

elevated 탭바

Surface

모달, 드롭다운, 토스트 등 서피스 컴포넌트용 그림자입니다.

dropdown

드롭다운 메뉴, 셀렉트 옵션

popover

팝오버, 툴팁

modal

모달, 다이얼로그

bottomSheet

바텀시트

drawer

사이드 드로어, 네비게이션

toast

토스트 메시지

Input Focus

입력 필드 포커스 링 그림자입니다.

inputFocus

입력 필드 포커스 링 (브랜드 컬러 기반)

inputError

입력 필드 에러 상태 포커스 링

shadowOffset: { width: 0, height: 4 }
shadowOpacity: 0.10
shadowRadius: 4
🤖Android
elevation: 3
